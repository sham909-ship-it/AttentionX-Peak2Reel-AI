# 🚀 AttentionX – Peak2Reel AI

Turn long videos into viral-ready short clips using AI.

---

## 🔥 Overview

AttentionX is an AI-powered platform that helps creators extract the **most engaging moments** from long-form videos and convert them into **high-performing short clips** for platforms like Instagram Reels, YouTube Shorts, and LinkedIn.

Instead of manually searching for highlights, AttentionX automatically:

* Detects key insights
* Scores clips based on virality
* Generates scroll-stopping captions
* Explains *why* a clip will perform well

---

## ✨ Key Features

### 🟡 Golden Nugget Score (Virality Predictor)

Each clip is scored (1–100) based on:

* Emotional impact
* Relatability

## ⚙️ How It Works

1. Upload a video file
2. AI analyzes content (simulated in demo)
3. Extracts key insights (chunks)
4. Scores each clip
5. Generates hooks & captions
6. Displays results with explanations

## 🚀 Running Locally

### 1. Clone the repo

```bash
git clone <your-repo-link>
cd AttentionX
```

### 2. Install dependencies

```bash
pip install -r requirements.txt
```

### 3. Run backend

```bash
uvicorn backend.main:app --reload
```

### 4. Open frontend

Open:

```
frontend/index.html
```
